Player avatars (referenced from poker-position-trainer.html and
poker-table-decision-trainer.html).

All files in this directory are local copies bundled with the project to avoid
hotlink fragility. WISDOM.md treats hotlinked named-player avatars as a recurring
trap (see "Локальные ассеты").

To refresh the two real photos, run from project root:

    curl -L -o assets/avatars/legenden.jpg "https://backing-funds.com/assets/img/team/IlyaLegenden13.jpg"
    curl -L -o assets/avatars/pavel-zanozin.jpg "https://blefach-poker.ru/wp-content/uploads/2024/10/pavel-zanozin-v-funfarm.jpg"

The trainer intentionally points to local files only and should never hotlink
these URLs directly. If real photos are not available, use stable local fallback
files instead of external URLs.
