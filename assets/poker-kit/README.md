# Poker Kit

Shared assets for poker trainers.

## Folders

- `decks/` - card themes, card rendering helpers, external deck manifests.
- `chips/` - reusable poker chips, dealer button, stack helpers.
- `layouts/` - page-level layout presets that combine decks, chips, and table rules.

## Current Decks

- `classic-english-local` - local realistic English-pattern card faces cached from Wikimedia Commons.
- `bicycle-rider-back` - Bicycle Rider style back image from Wikimedia Commons.
- `online-four-color` - online-style generated deck shared with the main hand trainer.
- `classic-generated` - generated classic pips/faces, useful when images should not load.

Keep new card skins or chip sets here instead of duplicating CSS across trainer pages.
